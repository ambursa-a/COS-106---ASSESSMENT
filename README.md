## Project Overview
This project is a fully functional, multi-page responsive website built as a personal academic portfolio and student management platform. It serves to showcase academic achievements, personal projects, and technical skills, while also providing an interactive academic planner to manage coursework and daily tasks. 

The site was designed adhering to modern web design principles and demonstrates core competencies in HTML5, CSS3, and Vanilla JavaScript.

---

## Features & Pages

*   **Homepage:** A welcoming landing page featuring a professional layout, personal photograph, brief biography, and clear navigation to other sections of the site.
*   **About Me:** A detailed profile outlining educational background, career aspirations, technical skills, and personal hobbies/interests.
*   **Projects Showcase:** A gallery of sample projects complete with descriptive text, screenshots, and external links. 
*   **Interactive Academic Planner:** A functional task management tool powered by JavaScript. Users can:
    *   Add new academic tasks.
    *   Mark tasks as completed.
    *   Delete tasks.
*   **Contact Page:** A fully validated contact form capturing the user's name, email, phone number, and message. 

---

## Technologies & Technical Highlights

### HTML5
*   **Semantic Structure:** Built using semantic HTML tags (`<header>`, `<nav>`, `<section>`, `<footer>`, etc.) for better accessibility and SEO.
*   **Rich Media:** Implementation of tables, lists, images, hyperlinks, and multimedia (audio/video).
*   **Forms:** Structured user inputs for the contact page.

### CSS3
*   **Layouts:** Utilizes **Flexbox** and **CSS Grid** to create complex, responsive layouts.
*   **Responsive Design:** Mobile-friendly interface that adapts seamlessly to desktop, tablet, and mobile screens.
*   **Styling:** External stylesheet featuring a consistent, visually appealing color scheme, custom navigation styling, and modern animations/transitions.

### JavaScript (Vanilla)
*   **DOM Manipulation:** Dynamic content updates for the Academic Planner page.
*   **Event Handling:** Interactive button clicks and form submissions.
*   **Form Validation:** Client-side checks on the Contact Page ensuring no fields are left blank, emails are formatted correctly, and phone numbers contain only digits.
*   **Data Structures:** Usage of arrays and functions to handle the task management logic.
